# Renomeador de Arquivos em Massa

Este projeto consiste em uma aplicação em Python que permite ao usuário renomear múltiplos arquivos simultaneamente de forma rápida e eficiente.

## Funcionalidades

- Seleção múltipla de arquivos para renomeação.
- Interface gráfica amigável para fácil utilização.
- Possibilidade de definir o diretório de destino para os arquivos renomeados.

## Como Usar

Para utilizar o renomeador de arquivos em massa, siga os passos abaixo:

1. Clone o repositório para a sua máquina local:
   ```
   git clone https://github.com/rodineyw/rename.git
   ```
2. Certifique-se de ter o Python instalado em sua máquina. Este programa foi testado com Python 3.10.
3. Navegue até o diretório do projeto e execute o arquivo principal:
   ```
   python app.py
   ```
4. Utilize a interface gráfica para selecionar os arquivos que deseja renomear e defina o novo nome base.
5. Escolha o diretório onde os arquivos renomeados serão salvos.

## Requisitos

- Python 3.10 ou superior
- Tkinter (normalmente vem instalado com o Python)
